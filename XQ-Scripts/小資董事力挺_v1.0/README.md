小資董事力挺 v1.0
Important

本腳本僅限已綁定 XQ 推薦碼 @GB 的使用者執行。

腳本介紹
本腳本用於尋找股價較容易參與，且近期董監持股出現增加跡象的股票，作為內部人信心觀察工具。

程式會綜合多項資料進行判斷，協助使用者快速縮小觀察範圍。

實際計算公式、判斷門檻與演算法細節不公開。

功能特色
觀察董監持股的近期變化
篩選內部人持股增加的標的
兼顧較容易參與的價格區間
輸出股價與董監持股變化供人工判讀
腳本下載
下載 小資董事力挺 v1.0

安裝方式
下載 .xsb 檔案
開啟 XQ 全球贏家
進入 XScript 編輯器
選擇「匯入」
選取下載的 .xsb 檔案
將腳本加入選股中心
使用適合的執行頻率進行篩選
使用權限
已綁定推薦碼 @GB 的帳號可以執行
未綁定推薦碼 @GB 的帳號無法執行
版本資訊
版本：v1.0
類型：XQ 選股腳本
適用頻率：日線，建議盤後執行
注意事項
本腳本提供的是條件篩選結果，不代表股票一定上漲。

建議搭配價格位置、成交量、產業趨勢、基本面與風險管理進一步判斷。

免責聲明
本腳本僅供研究、資料分析及教學用途，不構成任何投資建議。

使用者應自行判斷並承擔相關投資及交易風險。
