# VCP 波動收縮選股 v1.0

> [!IMPORTANT]
> 本腳本僅限已綁定 XQ 推薦碼 `@GB` 的使用者執行。

## 腳本介紹

本腳本用於尋找趨勢結構較強、近期價格波動逐步收斂，並伴隨成交量變化的股票。

程式會綜合判斷趨勢、價格強度、波動結構與量能狀態，協助使用者快速篩選可能處於整理階段的標的。

> 實際計算公式、判斷門檻與演算法細節不公開。

## 功能特色

- 篩選趨勢結構較強的股票
- 觀察價格波動是否逐步收斂
- 分析近期成交量變化
- 提供波動資料供使用者進一步判讀
- 適合搭配日線圖與基本面分析

## 安裝方式

1. 下載 `.xsb` 檔案
2. 開啟 XQ 全球贏家
3. 進入 XScript 編輯器
4. 選擇「匯入」
5. 選取下載的 `.xsb` 檔案
6. 將腳本加入選股中心
7. 使用日線頻率執行

## 使用權限

- 已綁定推薦碼 `@GB` 的帳號可以執行
- 未綁定推薦碼 `@GB` 的帳號無法執行

## 版本資訊

- 版本：v1.0
- 類型：XQ 選股腳本
- 適用頻率：日線

## 注意事項

本腳本提供條件篩選結果，不代表股票一定上漲。

建議搭配價格位置、成交量、產業趨勢、基本面及風險管理進一步判斷。
