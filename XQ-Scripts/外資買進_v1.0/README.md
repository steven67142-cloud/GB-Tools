外資買進 v1.0
Important

本腳本僅限已綁定 XQ 推薦碼 @GB 的使用者執行。

腳本介紹
本腳本用於觀察外資在一段期間內的累計買賣超狀況，協助篩選外資資金較明顯流入的股票。

程式會綜合多項資料進行判斷，協助使用者快速縮小觀察範圍。

實際計算公式、判斷門檻與演算法細節不公開。

功能特色
彙整近期外資買賣超資料
篩選外資累計買超較明顯的標的
輸出外資累計買超資料供排序與比較
適合搭配價格趨勢與產業題材進一步判讀
腳本下載
下載 外資買進 v1.0

安裝方式
下載 .xsb 檔案
開啟 XQ 全球贏家
進入 XScript 編輯器
選擇「匯入」
選取下載的 .xsb 檔案
將腳本加入選股中心
使用適合的執行頻率進行篩選
使用權限
已綁定推薦碼 @GB 的帳號可以執行
未綁定推薦碼 @GB 的帳號無法執行
版本資訊
版本：v1.0
類型：XQ 選股腳本
適用頻率：日線
注意事項
本腳本提供的是條件篩選結果，不代表股票一定上漲。

建議搭配價格位置、成交量、產業趨勢、基本面與風險管理進一步判斷。

免責聲明
本腳本僅供研究、資料分析及教學用途，不構成任何投資建議。

使用者應自行判斷並承擔相關投資及交易風險。
