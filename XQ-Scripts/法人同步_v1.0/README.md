法人同步 v1.0
Important

本腳本僅限已綁定 XQ 推薦碼 @GB 的使用者執行。

腳本介紹
本腳本用於觀察外資、投信與自營商在指定期間內是否呈現同步買超，協助找出法人資金方向較一致的股票。

程式會綜合多項資料進行判斷，協助使用者快速縮小觀察範圍。

實際計算公式、判斷門檻與演算法細節不公開。

功能特色
統計三大法人近期累計買賣超
篩選法人資金方向同步的標的
輸出法人合計與各法人資料
適合搭配趨勢、成交量與基本面進一步確認
腳本下載
下載 法人同步 v1.0

安裝方式
下載 .xsb 檔案
開啟 XQ 全球贏家
進入 XScript 編輯器
選擇「匯入」
選取下載的 .xsb 檔案
將腳本加入選股中心
使用適合的執行頻率進行篩選
使用權限
已綁定推薦碼 @GB 的帳號可以執行
未綁定推薦碼 @GB 的帳號無法執行
版本資訊
版本：v1.0
類型：XQ 選股腳本
適用頻率：日線
注意事項
本腳本提供的是條件篩選結果，不代表股票一定上漲。

建議搭配價格位置、成交量、產業趨勢、基本面與風險管理進一步判斷。

免責聲明
本腳本僅供研究、資料分析及教學用途，不構成任何投資建議。

使用者應自行判斷並承擔相關投資及交易風險。
